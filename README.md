# HiProxy
A proxy extension for Chrome

```
Based on Manifest V3 （MV3）

vue3 + wxt + ts


全新版本，基于谷歌扩展MV3规范开发，适用于Chrome浏览器新版本。

代码框架采用 vue3 + wxt + ts


使用方式：
1、进chrome 浏览器扩展，直达地址 chrome://extensions/
2、开启开发者模式
3、把这个 zip 文件直接拖进去，就可以了，不用解压。

```
